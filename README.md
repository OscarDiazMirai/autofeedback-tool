# AutoFeedback Web App

A powerful tool built with **React**, **Vite**, and **TypeScript** designed to perform automated feedback and validation across different sections of a website. 

The application streamlines the auditing process by verifying critical metadata and technical information to ensure consistency and correctness.

## 🚀 Purpose

The primary goal of this app is to provide **automated feedback** by scanning and reviewing specific data points within a web environment, such as:
- **IDs & Selectors:** Ensuring unique identifiers are correctly implemented.
- **URL Validation:** Checking link structures and protocols (e.g., HTTPS patterns).
- **Metadata Review:** Verifying SEO and technical information.
- **Data Consistency:** Cross-referencing information across different app modules.

## 🛠️ Tech Stack

- **Framework:** [React 18](https://reactjs.org)
- **Build Tool:** [Vite](https://vitejs.dev)
- **Language:** [TypeScript](https://typescriptlang.org) (Strict Mode)
- **State Management:** Functional Components with Hooks.
